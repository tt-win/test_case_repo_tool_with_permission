我需要做一個 Test Case 快速編輯模式

1. 在 Test Case Management 頁面，原本的「大量新增模式」按鈕，改名為「大量模式」
2. 「大量模式」按下後改成下拉式選單，提供兩個選項，一個是原本的「大量新增模式」，一個是新增的「大量編輯模式」
3. 「大量編輯模式」按下後，開啟「大量編輯模式」modal
4. 若已篩選 test case，則 modal 中只包含篩選過的 test case

大量編輯模式 Modal
介面類似 Excel / Google Sheet，需要有以下功能：
1. 寬度設定為瀏覽器寬度的 90%，需有最小寬度限制
2. Test Case Number，Title，TCG，優先級，Precondition，Steps，Expected Result 編輯需包含在內
3. 需要有類似 Excel / Google Sheet 的單一資料批次貼上模式，但僅限同類型欄位。例如複製一筆 Preconditon，我可以使用 shift + 點擊起點/終點，或是 command/ctrl + 點擊來批次多筆 test case 的 precondtion 來貼上
4. 需要有 Excel / Google Sheet 的拉動複製模式，也就是按某個欄位的右下角 + 鍵後往下拖動，資料將會複製到這些被拖動經過的欄位
5. 需要有 Search and Replace / Search and Replace All
6. 不需要支援 markdown preview，單純做文字編輯
